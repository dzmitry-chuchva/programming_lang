Данный файл передается лексическому анализатору flex для генерации лексического сканера токенов для синтаксического анализатора. Синтаксический анализатор вызывает генерируемую лексическим сканером функцию yylex каждый раз, когда ему необходимо получить очередной токен, исходя из грамматики.
