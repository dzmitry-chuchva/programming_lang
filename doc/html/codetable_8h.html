Файл содержит объявления функций, типов, переменных, используемых менеджером таблицы кода.

Функции

ctable_entry* add_code_entry (  symbol *  for_func,
code *  func_code
) 
 

Функция заносит в таблицу кода новую запись, содердащую имя функции и ее код.

Аргументы:
for_func символ функции
func_code сформированный код функции
Возвращает:
указатель на добавленный элемент

void clear_code_table (  ctable_entry *  table  ) 
 

Функция производит освобождение выделенной под таблицу кода памяти.

Аргументы:
table таблица кода

ctable_entry* ctable_tail (  ctable_entry *  table  ) 
 

Функция возвращает указатель на последний элемент таблицы кода.

Аргументы:
table указатель на элемент таблицы кода
Возвращает:
указатель на последний элемент таблицы кода

char is_defined (  ctable_entry *  table,
char *  name
) 
 

Функция производит поиск по твблице кода функции с заданным именем. Если функция существует в таблице кода, значит она определена. В данном случае на тип TYPE_DEFINED не всегда можно полагаться, так как он используется также в других целях.

Аргументы:
table таблица кода
name имя функции для поиска
Возвращает:
0 - функция не определена, 1 - функция определена.

void print_ctable (  ctable_entry *  table  ) 
 

Функция производит вывод кода фукнции на экран. Используется при отладке.

Аргументы:
table указатель на элемент таблицы кода
