Этот файл содержит фукнцию main, и описвает весь процесс компиляции: от запуска синтаксического анализатора до трансляции и компоновки полученного целевого кода.

Функции

int main (  int  argc,
char *  argv[]
) 
 

Функция main вызывается при старте приложения в языке C/C++

Аргументы:
argc количество переданных аргументов командной строки
argv[] массив аргументов командной строки
Возвращает:
в случае ошибки возвращается 1, иначе 0.

void static_error (  char *  tstr,
char *  format,
  ...
) 
 

Реализует вывод сообщений о семантических ошибках, обнаруженных в тексте программы в процессе синтаксически управляемой трансляции.

Аргументы:
tstr строка, содержащая тип ошибки
format формат-срока в стиле printf, но с ограниченным набором спецификаторов типа

void usage (   ) 
 

Функция выводит информацию о способе запуска компилятора.

Переменные

char compile_only = 0
 

Данный флаг устанавливается, если при обработке командной строки компилятора встретился флаг -с, который указывает компилятору сделать только проверку синтаксиса программы.

char error_flag = 0
 

флаг семантической ошибки в исходном тексте программы
