symbol* lookup (  char *  name  ) 
 

Функция производит поиск символа в таблице символов текущего контекста и возвращает указатель на него в случае успеха. Используется в лексическом сканере для определения необходимости установки нового символа либо использовании уже имеющегося. Если символ не найден в текущем контексте, производится поиск символа с таким именем в глобальном контексте. Если не найдено и в глобальном контексте, то возращает NULL.

Аргументы:
name идентификатор для поиска
Возвращает:
указатель в таблице символов на найденный символ, либо NULL, если символ не найден

Переменные

context_stack_member* context = &start_context
 

В силу того, что существует всего 2 контекста, максимальное количество элементов стека равно 2. Но, структура context_stack_member разрабатывалась с учетом того, что количество контекстов (вложенности процедур) неограничено. Поэтому, вы сможете легко добавить возможности описания переменных в блоках операторов (как в языке C++), например.

int next_label_no = 0
 

См. также:
install_label

int next_string_no = 0
 

См. также:
install_string

int next_temp_no = 0
 

См. также:
install_temp

context_stack_member start_context
