Содержит реализацию функций генератора целевого кода.

Функции

char* code2asm (  symbol *  global_table,
ctable_entry *  cod,
char *  str
) 
 

Функция производит перевод списка операторов трехадресного кода функции в код ассемблер, используя информацию в глобальной и локальной таблицах символов.

Аргументы:
global_table глобальная таблица символов
cod список операторов трехадресного кода функции
str строка, куда производить запись
Возвращает:
указатель на сформированную строку

char* floatop_prepare (  char *  str,
code *  c,
symbol *  global_table,
symbol *  table,
int  ret_type
) 
 

Функция формирует ассемблерный код, который производит необходимые преобразования физического стека программы для последующего вызова операции-функции над вещественными аргументами из библиотеки работы с вещественными числами.

Аргументы:
str строка, куда формировать
c трехадресная инструкция вещественного типа
global_table глобальная таблица символов
table локальная таблица символов
ret_type тип трехадресной инструкции
Возвращает:
указатель на сформированную строку

int generate_asm_code (  FILE *  where,
symbol *  data,
ctable_entry *  cod
) 
 

Функция производит обработку таблицы кода и таблицы символов, полученных в результате процесса синтаксически управляемой трансляции, и генерирует по ним целевой код ассемблер.

Аргументы:
where файл, куда генерировать
data таблица символов
cod таблица кода
Возвращает:
-1 - неудача, иначе - функция закончила свою работу без ошибок

char* local_addr (  symbol *  global_table,
symbol *  t,
symbol *  s
) 
 

Функция формирует строку, которая вычисляет адрес символа для передачи параметров функции по адресу и кладет его в физический стек программы.

Аргументы:
global_table глобальная таблица символов
t локальная таблица символов
s символ
Возвращает:
указатель на сформированную строку в статическом буфере (необходимо сразу использовать, так как следующий вызов функции запишет в него другое значение)

char* local_var (  symbol *  global_table,
symbol *  t,
symbol *  s
) 
 

Функция формирует строку в зависимости от того, в какой таблице (глобальной или локальной) находится символ. Код ассемблер будет представлять получение значения этого символа в виде [метка] (для глобальных) либо [ebp + число] (для локальных).

Аргументы:
global_table глобальная таблица символовы
t локальная таблица символов
s символ
Возвращает:
указатель на сформированную строку в статическом буфере (необходимо сразу использовать, так как следующий вызов функции запишет в него другое значение)

char* op2asm (  symbol *  global_table,
code *  c,
symbol *  table,
char *  str
) 
 

Функция получает элемент таблицы кода функции и переводит его в код ассемблер.

Аргументы:
global_table глобальная таблица символов
c трехадресная инструкция
table локальная таблица символов
str строка, куда производить запись
Возвращает:
указатель на сформированную строку

char* randomstr (  char *  s,
int  len
) 
 

Функция генерирует случайный набор символов, состоящий из заданного числа элементов. Используется для генерации имен меток в операторах цикла, ветвления.

Аргументы:
s строка, куда генерировать
len количество символов, которое необходимо сгенерировать
Возвращает:
указатель на сгенерированную строку
