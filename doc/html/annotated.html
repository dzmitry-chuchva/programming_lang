Компилятор MathC. Структуры данных

Структуры данных с их кратким описанием.
tag_codeструктура трехадресного оператора в списке операторов
tag_cstackструктура, предназначенная для реализации вложенности функций
tag_ctableэлемент таблицы кода
tag_fci_tтип стека вызовов функций в качестве параметров другой функции
tag_stack_tструктура данных стека
tag_symbolстуктура, описывающая символ в таблице символов
tag_valобъединение, служащее для хранения значения (может быть либо вещественным либо целым)
valptr_tобъединение, используемое для приведения типов symbol* и val_t
