Файл содержит объявления функций, переменных, типов, реализующих абстрактный стек, который может содержать элементы любого типа.

Функции

stack_t* create_stack (  void *  data,
int  el_size
) 
 

Функция выделяет память на первый элемент стека, заполняет его переданными значениями, устанавливает размер элемента стека.

Аргументы:
data информация нового элемента стека
el_size размер данных нового и последующих элементов стека
Возвращает:
указатель на созданный стек из одного элемента

stack_t* destroy_stack (  stack_t *  st  ) 
 

Функция освобождает память, выделенную под стек.

Аргументы:
st указатель на вершину стека
Возвращает:
возвращает NULL

stack_t* pop (  stack_t *  st  ) 
 

Функция производит выталкивание текущего элемента на вершине стека и его уничтожение.

Аргументы:
st вершина стека
Возвращает:
новая вершина стека

stack_t* push (  stack_t *  st,
void *  data,
int  el_size
) 
 

Функция создает новый элемент стека по переданными параметрам и кладет его на вершину стека.

Аргументы:
st указатель на вершину стека
data данные нового элемента
el_size размер данных нового элемента
Возвращает:
возвращает новый указатель на вершину стека

void* top (  stack_t *  st  ) 
 

Функция возвращает указатель на данные, записанные в текщей вершине стека.

Аргументы:
st вершина стека
Возвращает:
указатель на данные
